# MoneroMesh (monero-observer)

## About

MoneroMesh is an open-source observer designed to monitor and check pool and Monero blockchain statistics in real time.  
It focuses on P2Pool and Monerod nodes, providing miners and node operators with transparent and privacy-preserving data on network health, hashrate, block heights, and peer status.

## Features

- Real-time monitoring of P2Pool mini and main chains  
- Lightweight backend, suitable for running on low-resource devices like Raspberry Pi  
- Modern frontend dashboard built with React and Tailwind CSS  
- Fully open-source and privacy-first approach
